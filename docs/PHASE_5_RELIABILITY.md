# Фаза 5: Надежность и долговечность

Статус: Done

## Что реализовано

1. Добавлен набор автотестов критичных сценариев CLI в tests/test_time_tracker_cli.py.
2. Покрыты проверки устойчивости бизнес-правил:
   - старт/стоп сессии
   - запрет второй активной сессии
   - запрет stop без активной сессии
   - валидация интервала при edit
   - аудит правок в session_edits
3. Покрыты проверки календаря, summary, фильтра по заметке и экспортов CSV/JSON.
4. Добавлен таймзонный edge-case:
   - ввод времени с offset (+0300)
   - проверка нормализации в UTC для хранения.

## Запуск тестов

1. Полный прогон:
   - python3 -m unittest discover -s tests -p "test_*.py" -v
2. Один файл:
   - python3 -m unittest tests/test_time_tracker_cli.py -v

## Критерии завершения фазы 5

1. Критичные тесты проходят.
2. Таймзонные случаи не ломают хранение времени.
3. Проверены экспорты и отчетные команды.
